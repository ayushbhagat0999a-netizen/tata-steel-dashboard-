# # 📊 Tata Steel Financial Analysis Dashboard

## Overview
A technical financial analysis project bridging modern business management principles with data visualization. This interactive dashboard evaluates corporate fundamentals, demonstrating a strong capability to synthesize raw financial data into actionable equity research insights.

## 🚀 Key Features
* **Interactive UI:** Developed a TradingView-inspired financial dashboard to track Tata Steel's performance using HTML, CSS, and vanilla JavaScript.
* **Data Visualization:** Visualized 5-year historical data utilizing Chart.js to map trends in Revenue, EBITDA, Net Profit, and Debt Reduction.
* **Backend Logic Display:** Engineered a dedicated repository of MySQL queries to demonstrate backend data structuring, financial aggregation, and relational database logic.
* **Financial Rigor:** Analyzed key performance indicators, including geographic segment revenue and profitability margins, translating raw data into accessible business metrics.
* **Responsive Design:** Designed a responsive, multi-tab interface allowing seamless navigation between financial overviews, dynamic visual charts, and backend methodology.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Charting Library:** Chart.js
* **Database/Logic:** MySQL (Queries showcased within the UI)
